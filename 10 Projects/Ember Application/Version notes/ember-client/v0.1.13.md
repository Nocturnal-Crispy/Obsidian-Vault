# Ember Client v0.1.13 Release Notes

**Release Date:** 2026-03-17  
**Git Tag:** v0.1.13

## 🚀 Major Features

### Screen Sharing System
- **Complete screen sharing infrastructure** with modal picker and source selection
- **Multiple simultaneous screen shares** per channel support
- **Native audio capture** with Linux (PipeWire) and Windows (WASAPI) platform support
- **AudioWorklet pipeline** for low-latency audio processing
- **Screen share lifecycle management** with proper cleanup and state tracking

### Enhanced Video UI
- **Discord-style video interface** with improved layout and user experience
- **Native video pop-out windows** for flexible video management
- **Voice controls redesign** with 2-row layout and better organization
- **Screen share tile rendering** with immediate visual feedback

### Audio Capture System
- **Platform-specific audio capture addons** for Linux and Windows
- **PCM injector** for audio stream processing
- **Audio capture lifecycle integration** with screen sharing
- **Graceful degradation** to video-only when audio capture unavailable

## 🔧 Improvements

### App Lock Security
- **Enhanced PIN validation** with proper async handling
- **Conditional UI visibility** for app lock options when enabled
- **Security alerts** for missing PIN configuration
- **Improved test coverage** for app-lock-manager functionality

### Code Quality
- **Fixed async compatibility issues** in app-lock-manager tests
- **Removed duplicate WebSocket sends** in voice-ui-manager
- **Enhanced error handling** and logging throughout
- **Comprehensive test coverage** for all new features

## 📝 Technical Details

### New Components
- `src/renderer/components/screen-share-modal.ts`: Screen share source picker
- `src/main/audio-capture.ts`: Audio capture orchestration
- `src/renderer/audio/pcm-injector.ts`: Audio stream processing
- `src/preload/video-popout-preload.ts`: Video pop-out window support

### Enhanced Services
- `src/renderer/services/voice-service.ts`: Extended with screen sharing and audio capture
- `src/renderer/managers/voice-ui-manager.ts`: Redesigned UI with screen share controls
- `src/renderer/managers/app-state.ts`: Updated with screen share state management

### Native Addons
- `native/linux-audio-capture/`: PipeWire-based audio capture for Linux
- `native/windows-audio-capture/`: WASAPI-based audio capture for Windows

### UI Enhancements
- `src/renderer/styles/components/screen-share-modal.css`: Screen share modal styling
- `src/renderer/styles/components/voice.css`: Updated voice controls layout
- `src/renderer/video-popout.html`: Video pop-out window template

## 🔄 Upgrade Instructions

From v0.1.12:
1. Download the new version from releases page
2. Install using platform-specific installer:
   - Windows: Run `Ember Setup 0.1.13.exe`
   - Linux: Install `ember-client_0.1.13_amd64.deb` or use `Ember-0.1.13.AppImage`
   - Portable: Use `Ember-Portable.exe` (Windows) or AppImage (Linux)
3. Launch application - no configuration migration required
