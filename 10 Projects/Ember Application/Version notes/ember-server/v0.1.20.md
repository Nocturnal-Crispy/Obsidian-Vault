# Ember Server v0.1.20 Release Notes

**Release Date:** 2026-03-17  
**Git Tag:** v0.1.20

## 🚀 Features

### Direct Messaging Enhancements
- **feat(dm): device key enrollment** - Allow new devices to request the DM ember key
- **feat(dm): per-device key store** - Store one peer-box per requester device

## 🐛 Bug Fixes
- **fix(dm): fix key mismatch by having acceptor generate and distribute the shared ember key**

## 📝 Summary
This release focuses on improving direct messaging key management with device enrollment support and per-device key storage. The key mismatch issue has been resolved by having the acceptor generate and distribute the shared ember key.
