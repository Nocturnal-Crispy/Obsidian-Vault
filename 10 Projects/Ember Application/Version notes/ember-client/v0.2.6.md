# Ember Client v0.2.6 Release Notes

**Release Date:** 2026-03-29
**Git Tag:** v0.2.6

## 🚀 Features

### Build System
- **feat(build): native module rebuild script** - Added comprehensive native module rebuild script for better-sqlite3 cross-platform compilation with automatic fallback to node-gyp

## 🐛 Bug Fixes

### Build Process
- **fix(build): replace build:shared with build in secure-build script** - Fixed secure-build script referencing non-existent build:shared command

### Dependencies
- **chore(deps): update package versions and dependencies** - Updated package-lock.json and package.json with latest security patches

## 🔧 Refactoring

### Build Process
- **refactor(build): improve native module handling** - Enhanced build process to handle native module compilation more reliably across platforms
- **fix(build): secure-build script improvements** - Multiple fixes to the secure-build pipeline

## 📝 Summary
This release consolidates build system improvements from v0.2.3–v0.2.5 into a single properly published release with full build artifacts. Key changes include a new native module rebuild script for better-sqlite3, secure-build script fixes, and updated dependencies. This is the first release since v0.2.2 with complete distribution artifacts (AppImage, deb, Windows installers).

- No breaking changes in this release
