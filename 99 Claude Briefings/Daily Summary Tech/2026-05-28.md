# Tech Intelligence — 2026-05-28

## NVIDIA Rubin CPX: New GPU Class for Massive-Context Inference
**Heat: High | Momentum: NEW | Source: Tier 1 — Confirmed**

NVIDIA announced Rubin CPX, a purpose-built GPU for million-token context processing — targeting software coding and generative video workloads. This sits alongside the broader Rubin platform (336B transistors, up to 288GB HBM4, 22 TB/s memory bandwidth) shipping H2 2026. Rubin CPX availability is expected end of 2026.

**Strategic Impact:** Signals NVIDIA's pivot toward specialized inference silicon rather than one-size-fits-all training GPUs. Directly addresses the long-context bottleneck that limits agentic AI deployments.

## Custom AI Chips Outpace GPU Growth for First Time
**Heat: High | Momentum: RISING | Source: Tier 2 — Reputable**

Custom AI chip (ASIC) shipments from cloud providers are projected to grow 44.6% in 2026, nearly triple the 16.1% growth rate for merchant GPU shipments. Broadcom and Marvell control ~95% of the custom ASIC co-design market. Broadcom reported $8.4B in AI semiconductor revenue for Q1 FY2026 — up 106% YoY.

**Strategic Impact:** Structural shift away from merchant GPUs. Hyperscalers are increasingly designing their own silicon, potentially capping NVIDIA's data center growth ceiling long-term.

## Security Advisories

### Gitea Container Registry Vulnerability (CVE-2026-27771)
**Heat: Medium | Momentum: NEW | Source: Tier 2 — Reputable**

Unauthenticated attackers can pull private container images from Gitea deployments without credentials. Affects all versions prior to 1.26.2 — estimated 30,000+ deployments across healthcare, aerospace, retail, and ISPs. Undetected for ~4 years.

### Microsoft SharePoint RCE (CVE-2026-45659)
**Heat: Medium | Momentum: NEW | Source: Tier 1 — Confirmed**

CVSS 8.8 remote code execution vulnerability in SharePoint. No specialized conditions required for exploitation. Patch available.
