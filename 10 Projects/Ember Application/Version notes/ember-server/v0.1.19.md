# Ember Server v0.1.19 Release Notes

**Release Date:** 2026-03-17  
**Git Tag:** v0.1.19

## 🚀 Features

### Multiple Screen Shares Support
- **Extended participant state** for multiple simultaneous screen shares per channel
- **Enhanced participant metadata** with screen sharing flags and stream IDs
- **Improved voice participant broadcasting** with screen share information
- **Late joiner support** - participants joining mid-share receive current state

### Code Organization
- **Refactored voice handlers** from routes.go to dedicated voice_handlers.go file
- **Improved maintainability** by extracting 400+ lines of voice handler code
- **Enhanced test coverage** with comprehensive unit tests for voice handler functions

## 📝 Technical Details

### Server Changes
- `internal/transport/http/voice_handlers.go`: New file with extracted voice handler functions
- `internal/transport/http/routes.go`: Cleaner structure with voice handlers removed
- `internal/transport/http/voice_handlers_test.go`: Comprehensive test coverage
- Enhanced participant state management for screen share tracking

### API Enhancements
- Extended `sendVoiceParticipants` to include `screen_sharing` and `screen_stream_id` fields
- Improved `screen_share_start` and `screen_share_stop` handlers with state management
- Added `BuildVoiceParticipantEntry` helper for unit testing

## 🔄 Upgrade Instructions
