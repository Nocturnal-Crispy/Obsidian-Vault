# Tech Intelligence — 2026-06-15

## 1. Miasma Worm Compromises 73 Microsoft GitHub Repos in Supply Chain Attack
**Heat: Critical | Momentum: NEW | Source: Tier 1 (Confirmed)**

On June 5, a self-replicating supply chain malware dubbed "Miasma" compromised 73 Microsoft GitHub repositories across four organizations (Azure, Azure-Samples, Microsoft, MicrosoftDocs). The worm planted configuration files that execute credential-harvesting payloads when developers open repos in AI coding tools — Claude Code, Gemini CLI, Cursor, and VS Code. Stolen credentials were used to propagate to additional repositories. GitHub disabled all 73 repos within 105 seconds of detection. All repos have since been restored clean.

**Why it matters:** This is the first major supply chain attack specifically targeting AI coding agents at scale. The attack surface of AI-assisted development tools is now a confirmed vector for worm-class malware. Every organization using AI coding assistants needs to audit their tool configurations and repository trust chains.

## 3. Nvidia Enters the PC Market with RTX Spark Superchip
**Heat: High | Momentum: SUSTAINED | Source: Tier 1 (Confirmed)**

Nvidia's RTX Spark Superchip, unveiled at Computex alongside Microsoft, will debut in laptops and desktops from Dell, Lenovo, and HP this fall. Built on TSMC 3nm process. The Vera server CPU is already driving projected $20B in revenue this fiscal year. Jensen Huang frames the CPU market as a $200B industry opportunity. AMD, Intel, and Qualcomm shares dropped on the announcement.

**Why it matters:** Nvidia is now competing at every layer — data center GPUs, server CPUs, and consumer PCs. The x86 duopoly faces its most serious Arm-based challenge yet.
