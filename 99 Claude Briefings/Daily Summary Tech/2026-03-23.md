## Story 3: Critical Telnetd RCE — CVE-2026-32746 (CVSS 9.8)
**Heat:** Medium | **Momentum:** NEW | **Source Tier:** 1 (Confirmed)
**Categories:** Security

A critical pre-authentication vulnerability in telnetd (CVE-2026-32746, CVSS 9.8) enables unauthenticated root-level remote code execution via port 23. The flaw affects all versions through 2.7. A patch is expected by April 1, but no fix is available yet. This joins the Oracle Identity Manager RCE (CVE-2026-21992) reported yesterday as the second critical infrastructure vulnerability in 48 hours.

**Why it matters:** Any system running telnetd is vulnerable to full takeover without authentication. While telnet is legacy, many embedded systems and IoT devices still expose port 23. Organizations should disable telnetd immediately where possible.
