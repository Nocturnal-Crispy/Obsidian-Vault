# Ember Client v0.1.12 Release Notes

**Release Date:** 2026-03-16

## 🐛 Bug Fixes

- **Notification Sound**: Fixed notification sound to play correctly when channel element is not in DOM
- **DM Handling**: Improved direct message response handling and added polling support
- **WebSocket Notifications**: Enhanced WebSocket notification handler for better real-time updates

## 📝 Technical Details

- Fixed notification sound playback issues in certain DOM scenarios
- Enhanced DM response handling logic for better message delivery
- Added polling mechanism as fallback for real-time notifications
- Improved WebSocket notification handling for direct messages

## 🔄 Upgrade Instructions

- From v0.1.11: Download the new version from releases page
- No configuration changes required
