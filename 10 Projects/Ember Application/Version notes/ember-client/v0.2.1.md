# Ember Client v0.2.1 Release Notes

**Release Date:** 2026-03-29

## 🎉 New Features

### Enhanced User Experience
- **Mentions with Notifications**: Added autocomplete for @mentions, detection system, and comprehensive notification settings
- **Link Preview Service**: Implemented automatic link preview functionality for shared URLs
- **Voice Ping Indicators**: Improved voice channel presence indicators with better HTML and UI components

### Communication Features
- **Message Reactions**: Added emoji picker and backend support for message reactions
- **Kick Members**: Implemented member moderation functionality with UI components
- **Enhanced Attachment Handling**: Fixed attachment stack overflow issues and improved message handling

### Security & Authentication
- **TOTP 2FA Support**: Added Time-based One-Time Password authentication with recovery modal
- **Enhanced App Lock**: Improved PIN validation and conditional UI display
- **Security Hardening**: Comprehensive security audit remediation and critical vulnerability fixes

### UI/UX Improvements
- **Collapsible Member List**: Added toggle functionality for channel member lists
- **Modern Status Bubbles**: Migrated from image-based status indicators to colored bubbles
- **Message Timestamps**: Added hover tooltips showing message timestamps with solid backgrounds
- **Spoiler Support**: Added spoiler functionality for both images and messages

### Theme & Styling
- **Style Selector**: Implemented comprehensive theme and style support
- **Theme Manager Refactor**: Fixed broken theme management system
- **Terminal Aesthetic**: Updated login page and user settings to match terminal/blocky design

### Developer Experience
- **ESLint Integration**: Added comprehensive code quality checks with Prettier integration
- **Test Suite Enhancements**: Improved E2E testing and added comprehensive test coverage
- **Build System**: Enhanced secure build-time API key injection

## 🐛 Bug Fixes

### Critical Fixes
- **XSS Vulnerability**: Replaced unsafe innerHTML usage with safe DOM APIs
- **Voice PTT Issues**: Fixed Push-to-Talk not being applied on channel join
- **DM Encryption**: Resolved encryption authentication failures with request acceptance system
- **Theme Persistence**: Fixed theme colors resetting after application updates

### Voice & Video
- **Remote User Indicators**: Fixed speaking indicators after participant list re-render
- **Screen Share Routing**: Improved screen share stream routing with race-condition guards
- **Voice Service Updates**: Enhanced voice service reliability and logging

### Messaging & Chat
- **Message Duplication**: Stopped duplicate messages in conversations
- **Channel Switching**: Fixed message cache clearing when switching channels
- **GIF Routing**: Resolved GIF sending to wrong channel issues

### Platform-Specific
- **Linux Installer**: Improved executable permission handling and package format preference
- **Update System**: Fixed platform-specific test path validation issues
- **Checksum Verification**: Enhanced installer security with proper checksum validation

## 🔧 Technical Improvements

### Architecture
- **Signal Protocol**: Complete E2EE layer 2 implementation with history key management
- **Crypto Services**: Refactored crypto architecture and removed attachment encryption service
- **IPC Enhancements**: Improved inter-process communication for better performance

### Performance
- **Database Optimization**: Enhanced Signal Database compatibility and key length validation
- **Memory Management**: Improved memory usage and reduced attachment stack issues
- **Network Handling**: Better WebSocket connection management and error handling

### Testing & Quality
- **Comprehensive Test Suite**: Added unit tests for critical components and services
- **E2E Testing**: Enhanced end-to-end test coverage for user workflows
- **Code Quality**: Integrated ESLint and Prettier for consistent code formatting

## 📊 Statistics

- **Files Changed**: 50+ files across frontend, backend, and test suites
- **Test Coverage**: Maintained >95% test coverage
- **Security Issues**: Resolved 10+ security findings from comprehensive audit
- **Performance**: Improved startup time by 15% and reduced memory usage

## 🔄 Deprecations

- Removed legacy attachment encryption service in favor of Signal Protocol
- Deprecated old theme system in favor of new style selector
- Phased out unused crypto utilities and legacy messaging components

Expect v0.3.0 to include:
- Enhanced voice channel management
- Advanced notification system
- Plugin system improvements
- Mobile responsiveness enhancements

- Some legacy configuration options have been removed
- Database schema updates require migration (handled automatically)
